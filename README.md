## Overview

Snake Spectrum is an HTML arcade title that translates the classic snake formula into a smooth browser game. It is made for both mobile and desktop players, with responsive controls and a gameplay loop that is simple to start yet still supports extended sessions and repeat runs.

The release offers 120 structured levels plus an endless mode for players who prefer non-stop challenge. With polished movement, collectible items, unlockable visuals, and power-ups, it is a good match for anyone looking for a lightweight web game with plenty of variety.

## Usage

Launch the game in a modern browser, then choose level play or endless mode. On mobile, use touch controls; on desktop, use arrow keys and any other supported keyboard input.

Typical flow:
1. Start a level or begin an endless run.
2. Collect items and power-ups as you move.
3. Avoid obstacles and manage the increasing pace.
4. Visit the shop to unlock visual options.
5. Replay stages or switch to endless mode for a different challenge.
